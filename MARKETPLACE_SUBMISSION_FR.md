# Soumission ManualFinder sur Microsoft Marketplace (AppSource)

## 1. Prerequis
- Avoir un compte Microsoft Partner Center actif.
- Disposer des URLs publiques:
  - Support: `https://manualfinder.ai/contact`
  - Confidentialite: `https://manualfinder.ai/privacy`
  - Conditions: `https://manualfinder.ai/terms`
- Preparer les assets marketing (logo, screenshots, banniere).

## 2. Positionnement de l'offre
- Nom: ManualFinder
- Description courte: reconnaissance produit IA + acces instantane aux manuels
- Description longue: inclure cas d'usage fabricants, SAV, utilisateur final
- Categorie: Productivite / Support / IA (selon options Partner Center)

## 3. Verification technique
- Build local: `npm run build`
- Verification types: `npm run lint`
- Tester les sections du site:
  - Presentation
  - Fabricants
  - Public
  - Marketplace
  - Contact

## 4. Publication AppSource
1. Se connecter a Partner Center.
2. Creer une nouvelle offre Marketplace.
3. Completer les informations de listing (description, categories, images).
4. Ajouter les liens de conformite (support, privacy, terms).
5. Soumettre pour validation Microsoft.

## 5. Recommandation
- Ajouter rapidement les pages publiques `privacy` et `terms` sur votre domaine avant soumission.
